BetterConsole

NPM Package

This package adds useful info to the console replacing the habitual methods.

Usage

This package replaces the console methods, you require the package and use it like this


require('@memw/betterconsole')();


Methods

The methods are the console ones:


console.log(value);



console.trace(value);



console.debug(value);



console.info(value);



console.error(value);



console.warn(value);


if you don't want to add the info in a specific log there is an alternative method


console.normalLog(value);
