SimpleProgressBar

NPM package

SimpleProgressBar is a package that provides a simple progress bar for your processes.
With this you will easily be able to show the progress of your processes to the user of your project.

Image:



Usage


this package is a simple constructor

you require the package and use it like this


const simpleProgressBar = require('@memw/simpleprogressbar');



to use it you need to create a new instance of the class.


const progressBar = new simpleProgressBar(style);



the argument style is the style of the progress bar it is a string, example: "=>" or "##"

Methods


There are 3 methods you can use for this progress bar:

addToProgress lets you sum to the value of the progress bar, it doesn't return anything


progressBar.addToProgress(number);



resetProgress lets you reset the value of the progress bar, it doesn't return anything


progressBar.resetProgress();



and the last one is getProgress, it returns the string of the progress bar


progressBar.getProgress();
